# Как добавлять задания в Studify без Codex

Есть два способа.

## Способ 1. Быстрый импорт через приложение

1. Открой приложение.
2. Перейди в нижнем меню в `Профиль`.
3. Найди блок `Импорт заданий`.
4. Загрузи JSON-файл по структуре из `example-task-import.json`.
5. Новое задание появится в разделе `Задания`, а тест откроется отдельным экраном.

Импорт сохраняется в браузере через `localStorage`, поэтому он не пропадет после обновления страницы.

## Способ 2. Постоянно добавить задание в проект

Открой файл `studify-content.js` и добавь:

1. Новый тест в объект `tests`.
2. Новую кнопку задания в массив `tasks`.

Минимальный шаблон теста:

```js
newTestId: {
  id: "newTestId",
  subjectId: "russian",
  title: "Название теста",
  subtitle: "Русский язык",
  resultTitle: "Тест завершен",
  xpReward: 20,
  questions: [
    {
      question: "Текст вопроса",
      answers: ["Ответ 1", "Ответ 2", "Ответ 3"],
      correct: 0,
      hint: "Объяснение правильного ответа."
    }
  ]
}
```

Минимальный шаблон задания:

```js
{
  id: "new-test-task",
  day: "Сегодня",
  date: "2026-06-09",
  subjectId: "russian",
  title: "Название теста",
  xp: 20,
  testId: "newTestId"
}
```

## Список subjectId

- `russian` — Русский язык
- `basic-math` — Базовая математика
- `history` — История
- `social` — Обществознание
- `english` — Английский язык
- `geography` — География

## Важно

- `id` теста и `testId` задания должны совпадать.
- `correct` — это номер правильного ответа, начиная с нуля.
- Можно добавлять сколько угодно вопросов в `questions`.
- Можно добавлять сколько угодно заданий в `tasks`.
